# Diff command

## Requirements

| Name | Version |
|------|---------|
| go | >= 1.14 |

## Usage

```bash
./diff --help
Usage of ./diff:
  -i string
        Specify initial json (default "{}")
  -u string
        Specify updated json (default "{}")
  -json
        Output to json format
```

## Release

Update `version.txt` at root before releasing a new version (`version.txt` is a symlink of `module/version.txt`)
